Themes

Hexactyl ships with five built-in themes that can be switched instantly by both administrators and individual users. Each theme applies a complete color scheme across the entire panel, including the dashboard, server management pages, and admin areas.

Available Themes

Hexactyl (Orange)

The default theme. A bold orange and dark palette designed for high contrast and readability. Uses --primary: #f97316 as the accent color with a deep navy background.

--primary: #f97316;
--secondary: #fb923c;
--dark-bg: #0f0f23;
--sidebar-bg: #1a1a2e;

Midnight Blue

A cool blue theme for a calm, professional aesthetic. Ideal for gaming communities that prefer a darker, more subdued interface.

--primary: #3b82f6;
--secondary: #60a5fa;
--dark-bg: #0f172a;
--sidebar-bg: #1e293b;

Forest Green

A nature-inspired green theme. Clean and easy on the eyes, suitable for long administration sessions.

--primary: #22c55e;
--secondary: #4ade80;
--dark-bg: #0a1a0f;
--sidebar-bg: #14261a;

Sunset Orange

A warm, vibrant theme with reddish-orange tones. Provides a unique, energetic feel compared to the default.

--primary: #ef4444;
--secondary: #f97316;
--dark-bg: #1a0f0f;
--sidebar-bg: #2a1a1a;

Clean Light

A light theme with a white background and muted accents. Best for users who prefer a bright, traditional interface. Uses dark text on light backgrounds.

--primary: #2563eb;
--secondary: #3b82f6;
--dark-bg: #f8fafc;
--sidebar-bg: #ffffff;

Changing Themes (Admin)

Administrators can set the default theme for all users on the panel:

  1. Navigate to Admin Panel → Settings → Appearance.
  2. Select the desired theme from the Default Theme dropdown.
  3. Click Save Settings to apply.

This sets the theme for all users who have not chosen a personal theme. Individual users can override this by selecting their own theme in their account settings.

Changing Themes (User)

Each user can independently choose their preferred theme:

  1. Click your username in the top-right corner and select Account Settings.
  2. Scroll to the Appearance section.
  3. Select your preferred theme from the list.
  4. Click Update Preferences.

Your theme choice is saved per-account and persists across devices and sessions.

Custom CSS Support

Hexactyl allows administrators to inject custom CSS for further branding and customization. This is useful for adding logos, custom fonts, or adjusting spacing.

  1. Navigate to Admin Panel → Settings → Appearance.
  2. Locate the Custom CSS text area.
  3. Enter your custom styles. They will be loaded after the theme CSS, ensuring your overrides take precedence.
  4. Click Save Settings.

Custom CSS is stored in the database and applied on every page load. Changes take effect immediately without clearing caches.

Example: Custom Font

@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap');

body {
    font-family: 'Inter', sans-serif;
}

.sidebar-header h1 {
    font-weight: 700;
    letter-spacing: -0.5px;
}

Theme Variables Reference

All themes are defined using CSS custom properties. Below is the full list of variables you can override in your custom CSS or when creating a new theme:

Variable Default Value Usage
--primary#f97316Main accent color. Used for buttons, links, headers, and active states.
--secondary#fb923cSecondary accent. Used for hover states and gradients.
--dark-bg#0f0f23Page background color.
--sidebar-bg#1a1a2eSidebar background color.
--text-light#e0e0e0Primary text color.
--text-muted#a0a0a0Secondary and muted text color.
--border-color#333Border color for cards, tables, and dividers.
--code-bg#1e1e2eBackground color for code blocks and inline code.

To create a fully custom theme, override all variables in a CSS file and load it via the Custom CSS field or by adding a new stylesheet to resources/css/.
