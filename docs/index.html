What is Hexactyl?

Hexactyl is a free, open-source game server management panel built with PHP, React, and Go. Designed with security in mind, Hexactyl runs all game servers in isolated Docker containers while exposing a beautiful and intuitive UI to end users.

Originally based on the Pterodactyl project, Hexactyl has been enhanced with modern features, a fresh identity, and a focus on usability.

Key Features

Docker Isolation

Run game servers in completely isolated Docker containers for maximum security and resource management.

Multi-Node Support

Manage game servers across multiple physical machines from a single panel instance.

5 Beautiful Themes

Choose from Hexactyl, Midnight Blue, Forest Green, Sunset Orange, or Clean Light themes.

Server Templates

Create pre-configured templates for quick server deployment with one click.

Real-Time Console

Live server console with full terminal support using xterm.js.

Full REST API

Complete API for building custom integrations and automation.

Quick Start

Get up and running in minutes:

# Clone the repository
git clone https://github.com/Hexactyl-Projects/hexactyl.git

# Install dependencies
composer install
yarn install

# Configure environment
cp .env.example .env
php artisan key:generate

# Setup database
php artisan migrate
php artisan db:seed

# Build assets
yarn build:production

# Create admin user
php artisan p:user:make
